# Reset-Webflow CSS Reset

## Описание

Этот CSS-файл представляет собой комплексный сброс (reset) и нормализацию стилей для проектов на Webflow. Он обеспечивает единообразие базовых элементов, упрощает дальнейшую стилизацию и улучшает адаптивность и читаемость текста.

---

## Структура и важные блоки стилей

### 1. Базовые настройки страницы (`html`, `body`)

- **font-size:** адаптивный размер шрифта с помощью `clamp`, масштабируется от 14px до 20px в зависимости от ширины экрана.
- **scroll-behavior:** плавный скроллинг.
- **Сглаживание шрифтов:** свойства для оптимальной читаемости на разных браузерах.
- **min-height:** минимум высоты страницы занимает весь экран.
- **Цвета и отступы:** базовый сброс margin и padding, цвет текста и фона по умолчанию.

### 2. Контейнеры Webflow (`.w-layout-blockcontainer`, `.w-container`)

- Сброс max-width и отключение псевдоэлементов для контроля ширины и визуального представления.

### 3. Типографика

- Заголовки (`h1`–`h6`): сброс внешних отступов, наследование шрифтов и установка размера через rem с четкой иерархией.
- Параграфы, блоки цитат, адреса и списки очищены от дефолтных отступов и наследуют цвет.
- Для списков восстанавливается отступ слева для читаемости.

### 4. Ссылки и кнопки

- Сброс подчеркивания, установка цвета наследуемого от родителя.
- Плавные переходы при наведении с изменением цвета и текстового декорирования.
- Кнопки (`.w-button`, `input.w-button`) оформлены с базовыми цветами, фоном и padding, добавлен плавный переход и масштабирование при взаимодействии.

### 5. Формы и элементы управления Webflow

- Поля ввода и селекты (`.w-input`, `.w-select`) сброшены от дефолтных стилей, установлен прозрачный фон, рамки и padding.
- Убрана стандартная стрелка у select для кастомной стилизации.
- Чекбоксы и радиокнопки стилизованы с кастомными размерами, цветами и состояниями активного выбора (зелёный цвет).
- Убраны стандартные outlines при фокусе с возможностью добавить кастомные стили.

## Версии файла

В репозитории представлены две версии CSS reset-файла:

- **reset-webflow.css** — обычная, развернутая версия с читаемым и прокомментированным кодом. Предназначена для удобства изучения, правок и кастомизации.
- **reset-webflow.min.css** — минифицированная версия того же файла, оптимизированная по размеру для использования в продакшене. Минимальный вес помогает ускорить загрузку страницы.

Рекомендуется использовать **минифицированную версию** в рабочих проектах для повышения производительности, а **обычную версию** — при разработке и внесении изменений.
