# Reset-Webflow CSS Reset

## Описание

Этот CSS-файл представляет собой комплексный сброс (reset) и нормализацию стилей для проектов на Webflow. Он обеспечивает единообразие базовых элементов, упрощает дальнейшую стилизацию и улучшает адаптивность и читаемость текста.

В репозитории представлены два файла сброса:

### secure-reset.css

Содержит безопасные свойства, которые легко переназначаются через классы. Эти стили минимально вмешиваются в глобальные настройки и не влияют на стили, заданные с помощью свойства `all-*` в CSS. Благодаря этому вы получите гибкую основу без риска “сломать” существующую стилизацию при редактировании через Webflow Designer или код.

### -w-prefix-reset.css

Содержит небезопасные свойства, которые труднее переназначить, но включают удобные правила и полезные исправления. В этом файле:

- Значения `border` и псевдоэлемента `:after` у радиокнопок выставлены в `currentColor`, что позволяет автоматически наследовать цвет для более точной стилизации.
- Убраны все элементы, мешающие задавать `flex` на сайте (например, специфические отступы и фиксированные размеры).
- Сброшены стандартные отступы, паддинги, offsets и box-shadow, характерные для форм, что облегчает создание кастомных UI.
- В элементе `select` полностью убрана стандартная стрелка для последующей замены на уникальную.
- Для ссылок (`a`) с подчеркиванием задан `text-underline-offset: 3px` для удобочитаемого позиционирования линии.

## Структура и важные блоки стилей

### 1. Базовые настройки страницы (`html`, `body`)

- **Сглаживание шрифтов и оптимизация рендеринга:**  
  Используются свойства для улучшения читаемости текста и плавного отображения шрифтов в различных браузерах и операционных системах:  
  `-webkit-font-smoothing: antialiased;`, `-moz-osx-font-smoothing: grayscale;`, `text-rendering: optimizeLegibility;`, `-webkit-text-size-adjust: 100%;`.

### 2. Типографика

- Заголовки (`h1`–`h6`): сброс внешних отступов, наследование шрифтов и установка размера через `rem` с четкой иерархией.
- Параграфы, блоки цитат и адреса очищены от дефолтных отступов и наследуют цвет.
- Элементы списка (`li`): базовый сброс внешних и внутренних отступов, наследование размера шрифта.

### 3. Формы и элементы управления Webflow

- Поля ввода и селекты (`.w-input`, `.w-select`) сброшены от дефолтных стилей, установлен прозрачный фон, рамки и padding.
- Убрана стандартная стрелка у `select` для кастомной стилизации.
- Чекбоксы и радиокнопки стилизованы с кастомными размерами, цветами и состояниями активного выбора (зелёный цвет).
- Убраны стандартные outlines при фокусе с возможностью добавить кастомные стили.

## Версии файла

- **reset-webflow.css** — обычная, развернутая версия с читаемым и прокомментированным кодом, основанная на безопасных свойствах.
- **reset-webflow.min.css** — минифицированная версия того же файла, оптимизированная по размеру для продакшен-использования.
